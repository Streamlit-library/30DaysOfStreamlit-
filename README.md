# 30DaysOfStreamlit


# Day 1

- 1 Instalar conda
https://docs.conda.io/en/latest/miniconda.html 

- 2 Crear environment

~~~
conda create -n stenv python=3.9
~~~
- 3 Activar el environment

~~~
conda activate stenv
~~~
- 4 Instalar la librería Streamlit 

~~~
pip install streamlit
~~~
- 5 Ejecutar:

~~~
streamlit hello
~~~
